Planned Project: PRM

A PRM is like a CRM but it is for personal and professional networking. Compared to a CRM which is designed to help a user develop their relationships with customers to increase sales, a PRM is geared towards helping a user develop their personal and professional relationship networks in a strong and positive way in order to maximise benefit to the user and whomoever they decide to maintain a relationship with.

There is currently no GitHub repository associated with this project as it is still an idea in progress.
